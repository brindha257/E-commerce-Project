ShopZone – Modern E-Commerce Website
ShopZone is a simple and modern front-end e-commerce website built using HTML, CSS, and JavaScript.
It provides users with an interactive shopping experience — including product browsing, category filters, a functional cart, and checkout simulation.
--Project Overview
This project demonstrates how a responsive e-commerce layout can be built using only front-end technologies.
It includes a dynamic cart system, category filtering, and real-time product search — all handled via JavaScript without any backend.
--Features
    Responsive Navigation Bar with logo, search bar, and cart icon
    Live Product Search — filters products instantly as you type
    Category Filter Buttons — switch between Electronics, Fashion, and Grocery
    Add to Cart Functionality — items added dynamically to the cart sidebar
    Cart Sidebar — shows all selected items and total price
    Checkout Button — displays total cost and clears the cart
    Attractive UI Design — modern pastel theme with hover effects and shadows
    Fully Responsive Layout — works smoothly on desktop and mobile screens
---Technologies Used
  Technology	Purpose
HTML5	Structure of the website
CSS3	Styling, layout, and animations
JavaScript (ES6)	Dynamic functionality and interactivity
Google Fonts (Poppins)	Clean and modern typography
